# Descrições para Tooltips - TerraGIS

## MENU PROJETO

- **projeto-novo**: "Cria um novo projeto GIS com nome e fuso UTM definido"
- **projeto-salvar**: "Salva o projeto atual com todas as camadas e dados"
- **projeto-abrir**: "Abre um projeto GIS existente"

## MENU CONSTRUÇÃO

### POLÍGONO - TABELA
- **poligono-tabela-utm**: "Constrói um polígono a partir de coordenadas UTM digitadas em uma tabela"
- **poligono-tabela-geografica**: "Constrói um polígono a partir de coordenadas geográficas (latitude/longitude) digitadas em uma tabela"

### POLÍGONO - LISTA
- **poligono-lista-utm**: "Constrói um polígono a partir de uma lista de coordenadas UTM"
- **poligono-lista-geografica**: "Constrói um polígono a partir de uma lista de coordenadas geográficas"

### POLÍGONO - ÂNGULOS
- **poligono-azimute-distancia**: "Constrói um polígono a partir de azimute e distância entre vértices"
- **poligono-rumo-distancia**: "Constrói um polígono a partir de rumo e distância entre vértices"
- **poligono-sigef**: "Importa polígono do banco de dados SIGEF (Sistema de Gestão Fundiária)"

### POLILINHA - TABELA
- **polilinha-tabela-utm**: "Constrói uma polilinha a partir de coordenadas UTM digitadas em uma tabela"
- **polilinha-tabela-geografica**: "Constrói uma polilinha a partir de coordenadas geográficas digitadas em uma tabela"

### POLILINHA - LISTA
- **polilinha-lista-utm**: "Constrói uma polilinha a partir de uma lista de coordenadas UTM"
- **polilinha-lista-geografica**: "Constrói uma polilinha a partir de uma lista de coordenadas geográficas"

### POLILINHA - ÂNGULOS
- **polilinha-azimute-distancia**: "Constrói uma polilinha a partir de azimute e distância entre vértices"
- **polilinha-rumo-distancia**: "Constrói uma polilinha a partir de rumo e distância entre vértices"

### MÃO LIVRE
- **maolivre-poligono**: "Desenha um polígono à mão livre clicando no mapa para criar vértices"
- **maolivre-polilinha**: "Desenha uma polilinha à mão livre clicando no mapa para criar vértices"

## MENU EDIÇÃO

### VÉRTICE - ADICIONAR
- **vertice-adicionar-mapa**: "Adiciona um novo vértice clicando no mapa"
- **vertice-adicionar-coordenadas**: "Adiciona um novo vértice a partir de coordenadas digitadas"
- **vertice-adicionar-azimute**: "Adiciona um novo vértice a partir de azimute e distância"
- **vertice-adicionar-rumo**: "Adiciona um novo vértice a partir de rumo e distância"

### VÉRTICE - MOVER
- **vertice-mover-mapa**: "Move um vértice selecionado arrastando-o no mapa"
- **vertice-mover-coordenadas**: "Move um vértice selecionado para novas coordenadas digitadas"
- **vertice-mover-azimute**: "Move um vértice selecionado a partir de azimute e distância"
- **vertice-mover-rumo**: "Move um vértice selecionado a partir de rumo e distância"

### VÉRTICE - REMOVER
- **vertice-remover**: "Remove um vértice selecionado da geometria"

### VÉRTICE - RENOMEAR
- **vertice-renomear**: "Renomeia o identificador de um vértice selecionado"

### GEOMETRIA - MOVER
- **geometria-mover-mapa**: "Move uma geometria inteira arrastando-a no mapa"
- **geometria-mover-distancia**: "Move uma geometria inteira a partir de uma distância digitada"

### GEOMETRIA - COPIAR
- **geometria-copiar**: "Cria uma cópia de uma geometria selecionada"

### GEOMETRIA - ROTACIONAR
- **geometria-rotacionar-mapa**: "Rotaciona uma geometria clicando no mapa para definir o ângulo"
- **geometria-rotacionar-angulo**: "Rotaciona uma geometria a partir de um ângulo digitado"

### GEOMETRIA - CONVERTER
- **geometria-converter-polilinha-poligono**: "Converte uma polilinha aberta em um polígono fechado"

## MENU ANÁLISE

- **analise-poligono**: "Calcula e exibe informações de um polígono (área, perímetro, coordenadas dos vértices)"
- **analise-polilinha**: "Calcula e exibe informações de uma polilinha (comprimento, coordenadas dos vértices)"

## MENU MEDIDAS

- **medida-linha**: "Mede a distância entre dois pontos clicados no mapa"
- **medida-area**: "Mede a área de uma região desenhada no mapa"
- **medida-azimute**: "Mede o azimute (ângulo) entre dois pontos clicados no mapa"
- **medida-angulo**: "Mede o ângulo entre três pontos clicados no mapa"
- **medida-coordenada-vertice**: "Exibe as coordenadas de um vértice selecionado"

## MENU IMPRESSÃO

- **impressao-layout**: "Abre o editor de layout para configurar a impressão do mapa"

## PAINEL LATERAL - CAMADAS

- **camadas-toggle**: "Abre/fecha o gerenciador de camadas do projeto"
- **camadas-nova**: "Cria uma nova camada vazia no projeto"

## PAINEL LATERAL - BASE DO MAPA

- **basemap-seletor**: "Seleciona a camada de base do mapa (OpenStreetMap, satélite, etc.)"

---

**OBSERVAÇÕES:**
- As descrições devem ser breves mas completas
- Cada tooltip deve descrever a função principal da ferramenta
- Usar linguagem clara e acessível para usuários de GIS
