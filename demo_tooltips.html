Demonstração de Tooltips para TerraGIS

TOOLTIPS NATIVOS

Usa o tooltip padrão do navegador (title)

Características:
  • Aparência padrão do navegador
  • Simples de implementar
  • Aparência pode variar entre navegadores
  • Passe o mouse sobre os botões para ver

TOOLTIPS CUSTOMIZADOS

Usa CSS customizado com aparência profissional

Características:
  • Aparência customizada e profissional
  • Fonte tamanho 9pt exato
  • Consistente em todos os navegadores
  • Delay de 1,5 segundos
  • Passe o mouse sobre os botões para ver

QUAL VOCÊ PREFERE?

NATIVOS: Simples, mas menos controle visual
CUSTOMIZADOS: Profissional, com controle total sobre aparência
